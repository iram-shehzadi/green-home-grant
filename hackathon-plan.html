1Overview & the MVP floor

Build a digital service that replaces a 35-minute eligibility phone call. Citizens answer 5 questions, one per page, and learn instantly whether they qualify for the Green Home Grant, what for, and what to do next.

Win condition: judges reward a clean, complete, accessible service over a flashy incomplete one. Ship the six floor items below first — then build outward. Document your AI use as you go, not at the end.

The six mandatory items (the floor)

  • MVP 1Multi-page question flow — start page → 5+ question pages, one thing per page.
  • MVP 2Check-your-answers — summary list with a “Change” link for every answer.
  • MVP 3Result page — clear outcome: eligible / partial / not eligible.
  • MVP 4GOV.UK components — correct typography, spacing, heading hierarchy (use the existing App.css classes).
  • MVP 5WCAG 2.2 AA — labelled inputs, keyboard nav, 4.5:1 contrast, visible focus.
  • MVP 6AI usage log — AI_LOG.md, 3+ entries, 4 fields each, varied patterns.

2Our approach, in plain English

We chose “Vertical slice, then feature ownership.” Here is what that actually means, step by step:

  1. All four of us sit together for ~1 hour and build one thin path through the app: the start page → one question page → a results page with a fake (hardcoded) answer. We are not trying to finish the app — we are trying to agree how a page is built.
  2. We write down the rules we just discovered into a file called CLAUDE.md (how answers are stored, how pages talk to each other, which CSS classes to use, how errors look). This file is our shared rulebook.
  3. Then we split up. Each person owns one area (Flow, Logic, Review, Quality) and builds independently — but because we all follow the same CLAUDE.md, our work fits together with no surprises.
Why this beats “everyone grabs a page and goes”: the single biggest reason hackathon apps break is that people store answers differently and the data gets lost between pages. Building one slice together first means we only solve that problem once, together, before anyone works alone.
Plain-language glossary (open me if any term is unfamiliar)
Vertical slice
A thin end-to-end path through the app — start to finish — that proves the plumbing works, even if most features are missing or faked.
State / form data
The answers the user has given so far, kept in memory as the user moves between pages. We keep all of it in one object so nothing gets lost.
“Locking the state shape”
Agreeing the exact names and structure of where answers are stored (e.g. the key is propertyType, the value is a string like "house") so everyone reads and writes it the same way.
Component / page
A reusable piece of screen, written in React. Each question is its own page component.
Props
How a parent page passes data and functions down to a child page (e.g. passing the current answers and a “save this answer” function).
Pure function
A function that takes inputs and returns an output with no side effects — easy to test. Our eligibility logic is one of these.
CLAUDE.md
A plain-text rulebook in the project that both we and Claude read, so AI-assisted code follows our conventions automatically.

# Green Home Grant — Eligibility Checker

GOV.UK-styled React service that replaces a 35-minute eligibility phone call.
Citizens answer 5 questions (one per page) and get an instant eligibility result.

## Commands
- `npm run dev`   — start the Vite dev server
- `npm run build` — production build
- `npm test`      — run unit tests (add vitest first)

## Architecture (do not deviate without telling the team)
- ALL answers live in ONE `formData` object in `src/App.jsx`. Never keep answers in a page's own state.
  ```js
  const [formData, setFormData] = useState({
    propertyType: '', ownership: '', income: '', insulation: '', heating: ''
  });
  const updateField = (key, value) => setFormData(prev => ({ ...prev, [key]: value }));
  ```
- Every page receives `formData` and `updateField` as props from App.jsx.
- Eligibility logic is a PURE function in `src/utils/eligibility.js` — no React inside it.

## Answer vocabulary (use these EXACT string values)
- propertyType: house | bungalow | flat | maisonette | other
- ownership:    own | private-rent | housing-association | council
- income:       under-31000 | 31000-50000 | over-50000
- insulation:   both | wall | loft | none | dont-know
- heating:      gas | oil | electric | heat-pump | other

## Route order
/ (Start) -> /property-type -> /ownership -> /income -> /insulation -> /heating
  -> /check-answers -> /result

## Question-page pattern
Copy `src/pages/PropertyTypePage.jsx` — it is the reference. Every question page MUST:
- show a back link at the top that returns to the previous route
- put the question text in an 
 INSIDE the  (screen readers hear it once)
- render options with the govuk-radios classes, bound to formData via updateField
- on Continue: if no answer is selected, show the error summary (and focus it),
  show an inline error with the SAME wording, and prefix the page  with "Error: "
- on a valid answer, navigate to the next route

## Eligibility contract
checkEligibility(formData) -> {
  outcome: 'eligible' | 'partial' | 'not-eligible',
  funds: string[],            // e.g. ['insulation', 'heat-pump']
  fundingLevel?: string,      // 'full' | 'contribution' | 'voucher'
  survey?: boolean,
  reasons?: string[]
}

## GOV.UK conventions
- Reuse the classes already in `src/App.css` (govuk-*). Do NOT write new CSS.
- Errors: error summary (role="alert", "There is a problem") + inline govuk-error-message, same wording.
- Test every screen at 320px width, with keyboard only, with visible focus.

## Testing
- Use vitest. Eligibility tests go in `src/utils/eligibility.test.js`. Minimum 5 (we have 8 worked cases).

## AI_LOG (do this AS YOU GO)
- For every AI-assisted change, add an entry to `AI_LOG.md`:
  Date | Task | What AI Generated | What You Changed + Why.

export default function PropertyTypePage({ formData, updateField }) {
  const navigate = useNavigate();
  const [error, setError] = useState('');

  function handleContinue() {
    if (!formData[FIELD]) {
      const msg = 'Select the type of property you live in';
      setError(msg);
      document.title = `Error: ${QUESTION} - GOV.UK`;
      // focus the summary so screen readers and keyboard users hear the error
      setTimeout(() => document.getElementById('error-summary')?.focus(), 0);
      return;

    navigate(NEXT);

  return (
    <>
      <a href={PREV} className="govuk-back-link"
         onClick={(e) => { e.preventDefault(); navigate(PREV); }}>Back</a>

        <div className="govuk-error-summary" id="error-summary" tabIndex={-1} role="alert">
          <h2 className="govuk-error-summary__title">There is a problem</h2>
          <div className="govuk-error-summary__body">
            <ul className="govuk-list govuk-error-summary__list">
              <li><a href={`#${FIELD}`}>{error}</a></li>
            </ul>
          </div>
        </div>

      <div className={`govuk-form-group${error ? ' govuk-form-group--error' : ''}`}>

            <h1 className="govuk-heading-xl">{QUESTION}</h1>
          </legend>
          {error && (
            <p id={`${FIELD}-error`} className="govuk-error-message">
              <span className="govuk-visually-hidden">Error:</span> {error}
            </p>
          )}
          <div className="govuk-radios" id={FIELD}>
            {OPTIONS.map((opt) => (

}

      <pre><code id="eligText">export function checkEligibility({ propertyType, ownership, income, insulation, heating }) {
  // 1. Gate — private renters must apply via their landlord
  if (ownership === 'private-rent')
    return { outcome: 'not-eligible', funds: [], reasons: ['landlord-must-apply'] };

  // 2. What could be funded
  const needsInsulation   = insulation !== 'both';
  const insulationUnknown = insulation === 'dont-know';
  const canHeatPump       = heating !== 'heat-pump' && propertyType !== 'flat'; // flats: no external unit
  const funds = [];
  if (needsInsulation) funds.push('insulation');
  if (canHeatPump)     funds.push('heat-pump');

  // 3. Nothing left to fund
  if (funds.length === 0)
    return { outcome: 'not-eligible', funds: [], reasons: ['already-efficient'] };

  // 4. Income sets the funding level
  if (income === 'under-31000')
    return { outcome: 'eligible', funds, fundingLevel: 'full', survey: insulationUnknown };

  if (income === '31000-50000')
    return { outcome: 'partial', funds, fundingLevel: 'contribution', survey: insulationUnknown };

  // income === 'over-50000' — insulation not funded; heat-pump voucher only
  if (funds.includes('heat-pump'))
    return { outcome: 'partial', funds: ['heat-pump'], fundingLevel: 'voucher', survey: false };

  return { outcome: 'not-eligible', funds: [], reasons: ['income-too-high'] };
}</code></pre>

<script>
(function () {
  "use strict";

  var DEVS = ["dev1", "dev2", "dev3", "dev4"];

  // Strategy is fixed: A (vertical slice, then feature ownership).
  var STRATEGY = "vertical-slice-then-feature";

  // Suggested role -> dev mapping for Strategy A.
  var ROLE_BY_DEV = { dev1: "flow", dev2: "logic", dev3: "review", dev4: "quality" };
  var DEV_BY_ROLE = { flow: "dev1", logic: "dev2", review: "dev3", quality: "dev4" };

  var WORKSTREAMS = [
    { name: "Flow & state (Dev 1)", role: "flow", tasks: [
      { id: "app-state", label: "Lift formData into App.jsx + updateField callback" },
      { id: "q-pages", label: "Wire all 5 question pages (radios, value binding, Continue)" },
      { id: "navigation", label: "Back/Continue navigation with useNavigate across the flow" },
      { id: "start-page", label: "Start page: title, description, Start now button" }
    ]},
    { name: "Eligibility logic (Dev 2)", role: "logic", tasks: [
      { id: "elig-fn", label: "Write src/utils/eligibility.js (pure function)" },
      { id: "vitest-setup", label: "Add vitest to package.json + config" },
      { id: "elig-tests", label: "Write 8 unit tests from the worked cases" },
      { id: "result-page", label: "Result page: render outcome, funds, what-to-do-next" }
    ]},
    { name: "Review & validation (Dev 3)", role: "review", tasks: [
      { id: "check-answers", label: "Check-answers page: summary list + Change links" },
      { id: "validation", label: "Client-side validation: error summary + inline errors" },
      { id: "error-title", label: "Add 'Error: ' page-title prefix + focus error summary" }
    ]},
    { name: "Quality & accessibility (Dev 4)", role: "quality", tasks: [
      { id: "mount-chrome", label: "Mount PhaseBanner + GovukFooter in App.jsx" },
      { id: "wcag-pass", label: "WCAG 2.2 AA pass: labels, keyboard, contrast, focus" },
      { id: "responsive-320", label: "Test at 320px — no horizontal scroll" },
      { id: "a11y-statement", label: "Complete the accessibility statement page" },
      { id: "ailog-curate", label: "Curate AI_LOG.md — ensure 3+ varied entries" }
    ]}
  ];

  var AILOG_PATTERNS = [
    { id: "planning", label: "Planning", hint: "Using plan mode to scope the build before coding" },
    { id: "plan-mode-scaffolding", label: "Plan-mode scaffolding", hint: "Generating file/route skeletons from a plan" },
    { id: "multi-agent-critique", label: "Multi-agent critique", hint: "Having one agent review another's output" },
    { id: "refactor", label: "Refactor", hint: "Extracting/restructuring existing code" },
    { id: "test-generation", label: "Test generation", hint: "Generating unit tests from the ruleset" }
  ];

  var FIRSTPAGE_LABELS = {
    "property-type": "Property type",
    "insulation": "Insulation",
    "check-answers": "Check your answers"
  };

  var state = {
    firstPage: "property-type",
    names: { dev1: "", dev2: "", dev3: "", dev4: "" },
    taskOwners: {},
    taskPlanned: {},
    stretch: {},
    aiLog: {}
  };

  function devLabel(devId) {
    var n = state.names[devId];
    return n && n.trim() ? n.trim() : devId;
  }

  function ownerOptions(selected) {
    return DEVS.map(function (d) {
      var sel = d === selected ? " selected" : "";
      return '<option value="' + d + '"' + sel + '>' + escapeHtml(devLabel(d)) + "</option>";
    }).join("");
  }

  function escapeHtml(s) {
    return String(s).replace(/[&<>"']/g, function (c) {
      return ({ "&": "&", "<": "<", ">": ">", '"': """, "'": "'" })[c];
    });
  }

  function applyDefaults() {
    WORKSTREAMS.forEach(function (ws) {
      var owner = DEV_BY_ROLE[ws.role] || "dev1";
      ws.tasks.forEach(function (t) { state.taskOwners[t.id] = owner; });
    });
  }

  function renderTaskBoard() {
    var html = "";
    WORKSTREAMS.forEach(function (ws) {
      html += '<div class="workstream-head">' + escapeHtml(ws.name) + "</div>";
      ws.tasks.forEach(function (t) {
        html += '<div class="task-row">';
        html += '<div class="tlabel">' + escapeHtml(t.label) + "</div>";
        html += '<div><label class="vh" for="owner-' + t.id + '">Owner for ' + escapeHtml(t.label) + '</label>';
        html += '<select id="owner-' + t.id + '" data-task="' + t.id + '">' + ownerOptions(state.taskOwners[t.id]) + "</select></div>";
        var checked = state.taskPlanned[t.id] ? " checked" : "";
        html += '<div><input type="checkbox" data-plan="' + t.id + '" id="plan-' + t.id + '"' + checked + ' style="transform:scale(1.4);accent-color:var(--blue);">';
        html += '<label class="vh" for="plan-' + t.id + '">Plan ' + escapeHtml(t.label) + ' for day 1</label></div>';
        html += "</div>";
      });
    });
    document.getElementById("taskBoard").innerHTML = html;
  }

  function renderAiLog() {
    var html = "";
    AILOG_PATTERNS.forEach(function (p) {
      html += '<div class="card"><h4>' + escapeHtml(p.label) + "</h4>";
      html += '<p class="src">' + escapeHtml(p.hint) + "</p>";
      html += '<label class="fld" for="ai-' + p.id + '">Owner</label>';
      html += '<select id="ai-' + p.id + '" data-ai="' + p.id + '">';
      html += '<option value="">— unassigned —</option>' + ownerOptions(state.aiLog[p.id] || "");
      html += "</select></div>";
    });
    document.getElementById("aiLogBoard").innerHTML = html;
  }

  function refreshOwnerSelects() {
    document.querySelectorAll("select[data-task]").forEach(function (sel) {
      sel.innerHTML = ownerOptions(state.taskOwners[sel.getAttribute("data-task")]);
    });
    document.querySelectorAll("select[data-ai]").forEach(function (sel) {
      var cur = state.aiLog[sel.getAttribute("data-ai")] || "";
      sel.innerHTML = '<option value="">— unassigned —</option>' + ownerOptions(cur);
    });
  }

  function markFirstPage() {
    document.querySelectorAll(".opt").forEach(function (el) {
      el.classList.toggle("selected", el.getAttribute("data-firstpage") === state.firstPage);
    });
    var mid = document.getElementById("sliceMiddle");
    if (mid) mid.textContent = FIRSTPAGE_LABELS[state.firstPage] || "Property type";
  }

  function sortValue(v) {
    if (Array.isArray(v)) return v.map(sortValue);
    if (v && typeof v === "object") {
      return Object.keys(v).sort().reduce(function (acc, k) { acc[k] = sortValue(v[k]); return acc; }, {});
    }
    return v;
  }

  function buildOutput() {
    var assignments = {};
    DEVS.forEach(function (d) {
      assignments[d] = { name: devLabel(d), role: ROLE_BY_DEV[d] || "", tasks: [] };
    });
    Object.keys(state.taskOwners).forEach(function (taskId) {
      var owner = state.taskOwners[taskId];
      if (assignments[owner]) assignments[owner].tasks.push(taskId);
    });
    DEVS.forEach(function (d) { assignments[d].tasks.sort(); });

    var plannedDay1 = Object.keys(state.taskPlanned).filter(function (k) { return state.taskPlanned[k]; }).sort();
    var stretchGoals = Object.keys(state.stretch).filter(function (k) { return state.stretch[k]; }).sort();

    var aiLogPlan = {};
    DEVS.forEach(function (d) { aiLogPlan[d] = []; });
    Object.keys(state.aiLog).forEach(function (patternId) {
      var owner = state.aiLog[patternId];
      if (owner && aiLogPlan[owner]) aiLogPlan[owner].push(patternId);
    });
    DEVS.forEach(function (d) { aiLogPlan[d].sort(); });

    var out = {
      schemaVersion: "1.1",
      aiLogPlan: aiLogPlan,
      assignments: assignments,
      eligibilityRuleset: "concrete-v1",
      mobFirstPage: state.firstPage,
      plannedDay1: plannedDay1,
      strategy: STRATEGY,
      stretchGoals: stretchGoals,
      team: { members: DEVS.map(function (d) { return { id: d, name: devLabel(d) }; }) }
    };
    return JSON.stringify(sortValue(out), null, 2);
  }

  function init() {
    applyDefaults();
    renderTaskBoard();
    renderAiLog();
    markFirstPage();

    var defRadio = document.querySelector('input[name="firstpage"][value="' + state.firstPage + '"]');
    if (defRadio) defRadio.checked = true;

    document.querySelectorAll("#teamInputs input[data-dev]").forEach(function (inp) {
      inp.addEventListener("input", function () {
        state.names[inp.getAttribute("data-dev")] = inp.value;
        refreshOwnerSelects();
      });
    });

    document.querySelectorAll('input[name="firstpage"]').forEach(function (r) {
      r.addEventListener("change", function () {
        if (r.checked) { state.firstPage = r.value; markFirstPage(); }
      });
    });

    document.getElementById("taskBoard").addEventListener("change", function (e) {
      var t = e.target;
      if (t.matches("select[data-task]")) state.taskOwners[t.getAttribute("data-task")] = t.value;
      if (t.matches("input[data-plan]")) state.taskPlanned[t.getAttribute("data-plan")] = t.checked;
    });

    document.getElementById("aiLogBoard").addEventListener("change", function (e) {
      var t = e.target;
      if (t.matches("select[data-ai]")) state.aiLog[t.getAttribute("data-ai")] = t.value;
    });

    document.getElementById("stretchList").addEventListener("change", function (e) {
      var t = e.target;
      if (t.matches('input[type="checkbox"]')) state.stretch[t.value] = t.checked;
    });

    document.getElementById("genBtn").addEventListener("click", function () {
      document.getElementById("jsonOut").value = buildOutput();
      document.getElementById("copyStatus").textContent = "Generated.";
    });

    document.getElementById("copyBtn").addEventListener("click", function () {
      var ta = document.getElementById("jsonOut");
      if (!ta.value) ta.value = buildOutput();
      ta.select();
      copyText(ta.value, "copyStatus");
    });

    // Generic copy buttons (CLAUDE.md, skeleton, eligibility)
    document.querySelectorAll(".copybtn[data-copy]").forEach(function (btn) {
      btn.addEventListener("click", function () {
        var el = document.getElementById(btn.getAttribute("data-copy"));
        if (!el) return;
        var text = el.textContent;
        copyText(text, null, btn);
      });
    });
  }

  function copyText(text, statusId, btn) {
    function done(ok) {
      if (statusId) document.getElementById(statusId).textContent = ok ? "Copied to clipboard." : "Press Ctrl/Cmd+C to copy.";
      if (btn) {
        var orig = btn.textContent;
        btn.textContent = ok ? "Copied!" : "Select + Ctrl/Cmd+C";
        setTimeout(function () { btn.textContent = orig; }, 1500);
      }
    }
    if (navigator.clipboard && navigator.clipboard.writeText) {
      navigator.clipboard.writeText(text).then(function () { done(true); }, function () { done(false); });
    } else {
      var ok = false;
      try {
        var tmp = document.createElement("textarea");
        tmp.value = text; document.body.appendChild(tmp); tmp.select();
        ok = document.execCommand("copy"); document.body.removeChild(tmp);
      } catch (e) { ok = false; }
      done(ok);
    }
  }

  if (document.readyState === "loading") {
    document.addEventListener("DOMContentLoaded", init);
  } else {
    init();
  }
})();
